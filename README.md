<h1>Playstation Website Clone</h1>
<p>Este é um projeto de clone da página oficial da PlayStation que estou desenvolvendo como parte do meu aprendizado em desenvolvimento front-end. O projeto foi construído usando React.js e enfatiza a prática e aprimoramento das habilidades de desenvolvimento web, com foco especial na responsividade.</p>
